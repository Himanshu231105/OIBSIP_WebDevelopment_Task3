# 🌡️ OIBSIP Task 3 - Temperature Converter

## 📌 Objective
The objective of this project is to develop a temperature converter that converts values between Celsius, Fahrenheit, and Kelvin using JavaScript.

## 🚀 Project Description
This project is a web-based temperature converter that allows users to input a temperature value and convert it into different units. The application provides real-time results and displays values in all three units.

## ✨ Features
- Convert between Celsius, Fahrenheit, and Kelvin
- User-friendly interface
- Instant result display
- Conversion history tracking
- Input validation for accurate results
- Clean and responsive UI design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🔧 Steps Performed
1. Designed the user interface using HTML and CSS  
2. Created input fields and dropdown for unit selection  
3. Implemented conversion logic using JavaScript  
4. Displayed results dynamically using DOM manipulation  
5. Added extra features like history tracking and validation  

## 📊 Output
- Successfully converts temperature values accurately  
- Displays results in all three units  
- Maintains a history of conversions  
- Provides a smooth and interactive user experience
